# List Variables (keys) used in this plugin

1. ymc_fg_post_types
Список CPT. Стандартное сохранение в админке. Есть массив по умолчанию ['post']

2. ymc_fg_taxonomies
Есть массив по умолчанию []
Список выбранных таксономий. В БД храняться слаги таксономий.
При переключении CPT поле обнуляеться (устанавливаеться по умолчанию пустой массив [])

3. ymc_fg_terms
Есть массив по умолчанию []
Список выбранных термов. Стандартное сохранение в админке. В БД храняться IDs термов.
При переключении CPT поле обнуляеться (устанавливаеться по умолчанию пустой массив [])

4. ymc_fg_tax_attrs
Есть массив по умолчанию []
Список атрибутов для всех таксономий. Обновлятся будуть в админке с помощью AJAX.
При переключении CPT поле обнуляеться (устанавливаеться по умолчанию пустой массив [])

5. ymc_fg_term_attrs
Есть массив по умолчанию []
Список атрибутов для всех термов. Обновлятся будуть в админке с помощью AJAX.
При переключении CPT поле обнуляеться (устанавливаеться по умолчанию пустой массив [])

6. ymc_fg_tax_sort
Есть массив по умолчанию []
Список всех отсортированных таксономий в ручную. В БД храняться слаги таксономий.
Обновляеться при ручном перетаскивании в админке AJAX.
При переключении CPT поле обнуляеться (устанавливаеться по умолчанию пустой массив [])

7. ymc_fg_term_sort
Есть массив по умолчанию []
Список всех отсортированных термов. В БД храняться IDs термов.
Обновляеться при ручном перетаскивании в админке AJAX.
При переключении CPT поле обнуляеться (устанавливаеться по умолчанию пустой массив [])

8. ymc_fg_hierarchy_terms
Есть строка по умолчанию 'no'
Признак наличия иерархии термов в таксономиях (no | yes).

9. ymc_fg_selected_posts
Есть массив по умолчанию []
Выбранные посты. В БД храняться IDs термов. Добавляеться вручную в админке Add / Exclude Post(s).
При переключении CPT поле обнуляеться (устанавливаеться по умолчанию пустой массив [])

10. ymc_fg_excluded_posts
Есть строка по умолчанию 'no'
Признак исключения постов (no | yes).

11. ymc_plugin_legacy_is
Есть строка по умолчанию 'no'
Храняться значения в таблице wp_options,
Признак установки старого версии no | yes.

12. ymc_fg_tax_relation
Есть строка по умолчанию 'and'
Отношение между таксономиями and | or

13. ymc_fg_filter_options
Есть двухмерный массив по умолчанию. Применяеться для простых (одиночных) и комбинированогого фильтров.
Раздел: LAYOUTS -> FILTERS LAYOUT
[
    'tax_name' => 'category', // taxonomy name
    'filter_type' => 'default_filter', // default_filter | dropdown_filter etc.
    'location' => 'top' // top | left | right
]

14. ymc_fg_filter_hidden
Есть строка по умолчанию 'no'
Раздел: Appearance -> Filter Settings
Признак скрывать фильтр на фронтенде ( yes | no )

15. ymc_fg_filter_type
Есть строка по умолчанию 'default'
Тип фильтра ( default | dropdown_filter etc. )

16. ymc_fg_post_layout
Есть строка по умолчанию 'layout_standard'
Макет шаблона постов ( layout_standard | layout_masonry etc. )

17. ymc_fg_selection_mode
Есть строка по умолчанию 'single'
Режим выбора фильтров ( single | multiple )

18. ymc_fg_display_terms_mode
Есть строка по умолчанию 'selected_terms'
Режим отображения термов ( default | grouped | default_grouped )

19. ymc_fg_term_sort_direction
Есть строка по умолчанию 'asc'
Направление сортировки термов ( asc | desc | manual )

20. ymc_fg_per_page
Есть строка по умолчанию '5'
Количество постов на странице

21. ymc_fg_pagination_type
Есть строка по умолчанию 'numeric'
Тип пагинации ( numeric | infinite | loadmore )

22. ymc_fg_pagination_hidden
Есть строка по умолчанию 'no'
Признак скрывать пагинацию на фронтенде ( yes | no )

23. ymc_fg_term_sort_field
Есть строка по умолчанию 'name'
Поле сортировки термов ( name | count etc.)

24. ymc_fg_filter_all_label
Есть строка по умолчанию 'All'
Текст для кнопки "All" в фильтре

25. 'ymc_fg_filter_all_button' => [
        'category' => [
            'all_label'  => 'All',
            'is_visible' => true
        ]
    ]
Настройки для кнопки сброса фильтров "All"

26. ymc_fg_next_button_text
Есть строка по умолчанию 'Next'
Текст для кнопки "Next" в пагинации

27. ymc_fg_prev_button_text
Есть строка по умолчанию 'Prev'
Текст для кнопки "Prev" в пагинации

28. ymc_fg_load_more_text
Есть строка по умолчанию 'Load more'
Текст для кнопки "Load more" в пагинации

29. ymc_fg_post_image_size
Есть строка по умолчанию 'ymc_post_image_size'
Размер изображения поста ( full | medium | thumbnail | large )

30. ymc_fg_image_clickable
Есть строка по умолчанию 'no'
Признак кликабельности изображения ( yes | no )

31. ymc_fg_truncate_post_excerpt
Есть строка по умолчанию 'excerpt_truncated_text'
Признак обрезки контента поста ( excerpt_truncated_text | excerpt_first_block | excerpt_line_break )

32. ymc_fg_post_button_text
Есть строка по умолчанию 'Read more'
Текст для кнопки "Read more" в посте

33. ymc_fg_target_option
Есть строка по умолчанию '_blank'
Текст для кнопки "Read more" в посте

34. ymc_fg_post_excerpt_length
Есть строка по умолчанию '30'
Длина контента поста

35. ymc_fg_post_order
Есть строка по умолчанию 'date'
Направление сортировки постов ( asc | desc )

36. ymc_fg_post_order_by
Есть строка по умолчанию 'title'
Поле сортировки постов ( title | name | date | ID | menu_order | rand | comment_count | meta_value | meta_value_num | post__in )

37. ymc_fg_post_status
Есть строка по умолчанию 'publish'
Статус постов ( publish | future | draft | pending | private )

38. ymc_fg_no_results_message
Есть строка по умолчанию 'No results found'
Сообщение для отображения при отсутствии результатов поиска

39. ymc_fg_post_animation_effect
Есть строка по умолчанию 'none'
Эффект анимации поста ( none | fade | slide )

40. ymc_fg_post_display_settings
Есть массив по умолчанию []
Настройки отображения поста

41. ymc_fg_order_meta_key
Сортировка по meta key
Есть строка по умолчанию ''

42. ymc_fg_order_meta_value
Сортировка по meta value
Есть строка по умолчанию ''

43. ymc_fg_post_order_by_multiple
Есть массив по умолчанию []
Настройки сортировки постов

44. ymc_fg_enable_js_filter_api
Есть строка по умолчанию 'no'
Признак включения API фильтрации ( yes | no )

45. ymc_fg_post_columns_layout
Есть массив по умолчанию []
Настройки колонок постов

46. ymc_fg_post_grid_gap
Есть массив по умолчанию []
Настройки промежутков между колонками постов

47. ymc_fg_popup_enable
Есть строка по умолчанию 'no'
Признак включения всплывающих окон ( yes | no )

47. ymc_fg_popup_settings
Есть массив по умолчанию []
Настройки всплывающих окон
