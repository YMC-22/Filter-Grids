# Upgrade Notice — Filter & Grids

**Version:** 3.1.0  
**Release date:** 2025-09-10

## Important Information for Upgrade

Please read this document carefully before updating to the latest version of **Filter & Grids**.  
This update includes major changes to the plugin structure and functionality.

---

## ⚠️ What You Need to Know

- **Backup First**: Always create a full backup of your WordPress site (files + database) before upgrading.
- **Compatibility**: This version may not be compatible with older versions of WordPress (requires WP 5.8+).
- **Customizations**: If you have modified plugin files directly, those changes will be lost after the update.
- **Templates**: Custom grid templates may need adjustments to match the new layout engine.
- **Deprecated Functions**: Some old functions have been removed or replaced. Refer to the developer documentation.

---

## ✅ Recommended Steps

1. Backup your site (database + files).
2. Test the update on a staging site before applying it to your live site.
3. Update all shortcodes to ensure compatibility with new parameters.
4. Review your filters and grids after upgrade to ensure they display correctly.
5. Clear any caching (plugin + server-side) after the update.
